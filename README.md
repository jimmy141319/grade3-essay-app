# 三年級作文小幫手

這是一個適合國小三年級使用的作文練習小 APP。

## 功能

- 作文題目抽籤
- 四段式作文架構
- 作文提示句
- 常見三年級作文題型
- 練習輸入區
- 產生完整作文
- 複製作文
- 字數統計
- 手機版自適應畫面

## GitHub Pages 使用方式

1. 到 GitHub 建立新的 Repository
2. 上傳 `index.html` 和 `README.md`
3. 到 Repository 的 Settings
4. 找到 Pages
5. Source 選擇 `Deploy from a branch`
6. Branch 選擇 `main` / root
7. 儲存後等待 GitHub 產生網址

## 建議 Repo 名稱

`grade3-essay-app`

也可以取名：

- `essay-helper`
- `grade3-writing-helper`
- `kids-writing-app`
